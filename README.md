# 🌱 AI Plant Disease Detection

## 📌 Project Overview

AI Plant Disease Detection is a web application developed using **Python** and **Streamlit**. The system helps identify plant diseases from uploaded leaf images and provides AI-based recommendations such as symptoms, causes, prevention methods, and treatment suggestions.

---

## 🚀 Features

* Upload plant leaf images
* Detect plant diseases
* Display disease name
* AI-powered disease recommendations
* User-friendly Streamlit interface

---

## 🛠️ Technologies Used

* Python
* Streamlit
* Hugging Face Transformers
* Pillow (PIL)
* Git & GitHub

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/AI-Plant-Disease-Detection.git
```

Go to the project folder:

```bash
cd AI-Plant-Disease-Detection
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
